# Examples — Case Study Round

Three worked cases, each covering a different format. Every example walks through the full template with realistic content, trade-off narration, and a measurement close.

| Example | Format | Scenario |
|---------|--------|----------|
| [retail-support-automation.md](retail-support-automation.md) | Live case (45 min) | Retailer reducing support costs with AI |
| [churn-prediction-takehome.md](churn-prediction-takehome.md) | Take-home (3-hour timebox) | 100K rows churn data |
| [rag-pipeline-defense.md](rag-pipeline-defense.md) | Defense round | Defending a RAG pipeline design under challenge |

## How to use these

- **Before practice:** read the example once to internalize the template rhythm.
- **During drills:** close the file, pick a prompt, run the template out loud. Then compare.
- **Defense prep:** give the design to a peer and ask them to reproduce the challenge questions in [rag-pipeline-defense.md](rag-pipeline-defense.md).
