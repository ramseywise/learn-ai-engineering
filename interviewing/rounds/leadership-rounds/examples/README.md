# Leadership Round Worked Examples

Three worked examples covering the most common high-stakes moments in leadership rounds.

| File | What it demonstrates |
|------|---------------------|
| `cto-judgment-thread.md` | A live build-vs-buy discussion about model serving — how to hold a position, update on evidence, and show calibrated judgment |
| `90-day-sketch.md` | A fully worked 90-day plan for joining an AI startup — specific, credible, not hero mode |
| `reverse-interview.md` | Using the reverse-interview question bank to actually evaluate a startup — signals decoded in real-time |

These are realistic representations of how these conversations go, including pushback and pivots. Study the structure as much as the content.
