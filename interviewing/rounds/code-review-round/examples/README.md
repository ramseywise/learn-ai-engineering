# Code Review Round — Worked Examples

Each file covers a study guide topic area with 2-3 diffs and model reviews. Practice by reading the diff first, writing your own review, then comparing.

| Topic | File | Focus |
|-------|------|-------|
| Security & trust boundaries | [security.md](security.md) | Auth checks, injection, agent tool permissions, trust separation |
| Correctness & interfaces | [correctness.md](correctness.md) | Schema drift, swallowed errors, tool contract mismatches |
| Conventions & tunables | [conventions.md](conventions.md) | Hardcoded thresholds, prompt placement, config drift |
| AI-generated & LLM code | [ai-generated.md](ai-generated.md) | Hallucinated APIs, plausible-but-wrong logic, convention drift |
