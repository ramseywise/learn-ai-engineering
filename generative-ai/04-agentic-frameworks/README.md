## What it is

This pillar holds framework reference material: comparison documents, feature-parity
analyses, and selection guides for the major agentic frameworks. Where
[03-agentic-foundations/](../03-agentic-foundations/README.md) is about learning frameworks
through courses, this pillar is about choosing and configuring the right framework for a
given problem.
